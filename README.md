# Lab Multivariado

Sitio web de simuladores interactivos para el curso Cálculo Multivariado.

## Simuladores incluidos

- Simulador de gradiente.
- Simulador de integración múltiple y cambios de coordenadas.
- Enlace a simulador externo de Lagrange.

## Rutas

- Página principal: `/`
- Gradiente: `/simuladores/gradiente/`
- Integración múltiple: `/simuladores/integracion/`
